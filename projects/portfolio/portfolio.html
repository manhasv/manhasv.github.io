Description

A 22-year-old man too tired of job applications decided to do something useful. He chose to tackle the boss anyone asks him to do: a portfolio.

Actually though, this is nicer than I thought. I have this manhasv.github.io website since 2022 but I ditched it to do other stuff. Now it just dawned on me that I can just use it as a portfolio, so I decided to make it more professional and showcase my work and projects.

Job Logs: Getting Started (9/7/2025)

I started by buying the domain name on NameCheap.com for $6.69 (nice). While I was thinking about renting a server, I realized I could simply host it on github.io for free. Added a bunch of basic stuff, it looks a bit soulless right now but it's WIP. Worst thing so far is that I decided to use Tailwind CSS for styling, which I didnt use before so uhh.
